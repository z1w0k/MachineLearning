# Multi-Agent LLM Game Simulation: Mafia

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue.svg)](https://www.python.org/)
[![LLM](https://img.shields.io/badge/LLM-YandexGPT%20%7C%20OpenAI-green.svg)]()

> **О проекте:** Разработка симулятора классической психологической игры «Мафия», где роли игроков исполняют автономные ИИ-агенты на базе больших языковых моделей (LLM). 

Проект демонстрирует навыки оркестрации мультиагентных систем, обхода ограничений (Safety Filters) и парсинга неструктурированного естественного языка в строгие JSON-структуры.

## Ключевые особенности

* **Автономные агенты с изолированной памятью:** У каждой нейросети своя роль, характер и скрытая от других участников «память».
* **Централизованный Game Master:** Полностью автоматизированный ведущий (оркестратор), который управляет фазами игры (День/Ночь), очередью выступлений и детерминированно подсчитывает голоса.
* **Строгая типизация данных:** ИИ-агенты общаются через жесткий JSON-формат (`thoughts`, `chat`, `suspect`, `vote`). Встроенный парсер перехватывает ошибки и галлюцинации моделей.
* **Обход Safety Filters:** Промпт-инжиниринг с использованием «эзопова языка», позволяющий моделям (особенно YandexGPT) играть в Мафию, не срабатывая на фильтры цензуры и жестокости.
* **Интерактивный режим отладки:** Пошаговый CLI-интерфейс (через `Enter`) для контроля игры в реальном времени и анализа скрытых "мыслей" ИИ.

## Стек технологий

* **Язык:** Python 3 (ООП)
* **Интеграция ИИ:** Yandex Cloud API (YandexGPT), поддержка OpenAI API / OpenRouter / GigaChat
* **Инженерия данных и библиотеки:** `requests`, `python-dotenv`, `collections.Counter`, валидация JSON
